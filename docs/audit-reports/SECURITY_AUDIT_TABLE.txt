VERIFICATION COMMANDS
================================================================================

Check if .env is in git history:
  git log --all --full-history -- .env

Check if secrets/ is in git history:
  git log --all --full-history -- secrets/

Search for API keys in current code:
  grep -r "api_key\|private_key\|secret" --include="*.py" --include="*.js" \
    --exclude-dir=venv --exclude-dir=node_modules

Search for secrets in git history:
  git log -p | grep -i "api_key\|private_key\|secret" | head -50

Verify .env is in .gitignore:
  grep "^\.env$" .gitignore

Verify secrets/ is in .gitignore:
  grep "^secrets/$" .gitignore
